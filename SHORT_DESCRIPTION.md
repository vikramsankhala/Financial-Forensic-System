# Short Repository Description (for GitHub/GitLab)

**Production-Grade Real-Time Financial Fraud Detection & Forensics Platform**

A comprehensive fraud detection system combining ML-based anomaly detection (PyTorch autoencoder) with rule-based compliance checks and forensic case management. Built with FastAPI + Next.js for financial crime investigation teams.

**Features:** Real-time transaction scoring, automatic case creation, entity network visualization, full audit trails, role-based access control.

**Tech:** FastAPI, PostgreSQL, PyTorch, Next.js 14, TypeScript, Material UI, Docker, Fly.io

Perfect for financial institutions and forensic audit teams requiring real-time fraud monitoring and investigation workflows.
